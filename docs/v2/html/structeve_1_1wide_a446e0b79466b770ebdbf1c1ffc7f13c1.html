◆ slice() [2/2]

template<typename Type , typename Cardinal >
template<std::size_t Slice>
auto eve::wide< Type, Cardinal >::slice ( slice_t< Slice >  s) const
inline

Return the upper or lower half-sized slice of a eve::wide. Does not participate in overload resolution if Cardinal::value == 1.

See also
eve::upper_
eve::lower_
Parameters
sA tag indicating which slice is required

Example:

See it live on Compiler Explorer

#include <eve/wide.hpp>
#include <iostream>
int main()
{
// Generates the wide [1 ... N]
eve::wide<int> r = [](auto i, auto c) { return i + 1;};
std::cout << r << "\n";
auto lo = r.slice(eve::lower_);
auto hi = r.slice(eve::upper_);
std::cout << lo << " " << hi << "\n";
}
